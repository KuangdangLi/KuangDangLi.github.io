sync修饰符#

子组件尝试改变一个prop属性值并将其同步到父组件时，开发者可以使用sync修饰符。 代码实例

Vue.component("Child",{
    template: `
    <div class="child">
    {{money}}
    <button @click="$emit('update:money', money-100)">
      <span>花钱</span>
    </button>
  </div>
    `,
    props: ["money"]
})
new Vue({
    components: { Child: Child },
    template:`
    <div class="app">
    App.vue 我现在有 {{total}}
    <hr>
    <Child :money.sync="total"/>
  </div>
  `,
  data() {
    return { total: 10000 };
  }
})

子组件注册一个点击事件update:money，传出参数money-100，父组件使用.sync修饰符监听事件，并拿到子组件传出的参数并赋值给子组件的外部属性money。

//.sync修饰符相当于封装了如下操作
<Child :money="total" v-on:update-money="total = $event"/>
