组件的概念#

组件，从概念上类似于 JavaScript 函数。它接受任意的入参（即 “props”），并返回用于描述页面展示内容的 React 元素。

组件与元素的区别#

元素是一个React元素，而组件是一个返回React元素的函数；元素的名字一般以小写字母开头，而组件的第一个字母通常大写。

//定义一个简单的组件
const Div = ()=> <div className='hello'>hello World</div>
//定义一个简单的元素
const div = <div className='hello'>hello World</div>

组件的种类#

组件分为函数组件和类组件。

//函数组件
function Welcome(props) {
    return <h1>hello,{{props.name}}</h1>
}
//使用方法
<Welcome name='frank'/>
//函数组件
class Welcome extends React.Component{
    render(){
        return <h1>hello,{{this.props.name}}</h1>
    }
}
//使用方法
<Welcome name='frank'/>

JSX标签的翻译逻辑:

<div/ >会被翻译成c('div');

<Welcome />会被翻译为React.createElement(Welcome);

React.createElement():

如果传入一个字符串‘div’，则会创建一个div;

如果传入一个函数，则会调用该函数，获取其返回值;

如果传入一个类，则会new类(执行constructor)，获取一个组件对象，然后调用render方法，获取其返回值。

props#

当React元素为用户自定义组件时，它会将JSX所接收的属性(attributes)以及子组件(children)转换为单个对象传递给子组件，这个对象被称为"props"。

例如，下面这段代码会在页面上渲染 “Hello, React”：

function Welcome(props){
    return <h1>hello,{{props.name}}</h1>
}
const root = ReactDOM.createRoot(document.getElementById('root'));
const element = <Welcome name='React'/>;
//props为:{name:'React'}
root.render(element)

需要注意的是，React组件无论是使用函数声明还是通过 class 声明，都绝不能修改自身的 props。

state#

类组件的state#

  1. state初始化
    在构造函数中初始化state 
    class Welcome extends React.Component{
 constructor(){
     super();
     this.state = {
         n:0
     }
 }
}
  2. 读state
    this.state可读state对象的数据 
    class Welcome extends React.Component{
constructor(){
    super();
    this.state = {
        n:0
    }
}
render(){
    <div>n:{{this.state.n}}</div>
}
}
  3. 写state
    React组件中的state对象不可直接修改。 
    add(){
     this.state.n+=1;
}
    这种写法是不被允许的。有两点原因：第一，React不像Vue一样直接劫持监听数据，做不到根据数据变化主动更新UI；第二，React不鼓励在state对象自身上直接进行修改，若要变更数据请生成新的state对象。 正确的写state的姿势:
    • 使用this.setState来主动通知React修改数据。
    • 用新生成的对象代替原本的state。
    add(){
    this.setState({n:this.state.n+1})
}
    需要注意的是，React提供的this.setState是一个异步函数，一般使用函数来充当它的参数，而不是直接使用数据对象。 
    add(){
    this.setState((state)=>{n:this.state.n+1})
}

函数组件的state#

React提供React.useState()来初始化函数组件的state。

  • useState方法唯一的参数就是初始state。不同于类组件，我们可以按照需要使用数字或字符串对其进行赋值，而不一定是对象。如果开发者需要在state中存储两个不同的变量，只需要调用useState两次即可。
  • useState的返回值为当前state以及更新state的函数。
const Hello = ()={
    const [n,setN] = React.useState(0)
}

state合并#

  • 当state为只有一层的复杂对象时:

    类组件使用setState，React 会把你提供的对象合并到当前的 state；

    函数组件setN不会自动合并需要利用 ... 操作符等操作来手动合并。

  • 当state对象有多层时，类组件与函数组件都不会提供自动合并需要手动操作。
